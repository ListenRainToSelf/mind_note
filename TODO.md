## 当前任务详情

### Phase 1.1：`main.py` 应用入口

```python
# 预期功能
- 读取 config.json
- 调用 API.init() 完成后端初始化
- 启动 Flask 开发服务器
- 注册蓝图，绑定路由
```

### Phase 1.2：Flask 路由

```text
POST /api/login         → network_API.login()
POST /api/register      → network_API.register()
POST /api/logout        → network_API.logout()
POST /api/notes         → 创建笔记
GET  /api/notes         → 获取笔记列表
GET  /api/notes/<id>    → 读取笔记内容
PUT  /api/notes/<id>    → 更新笔记
DELETE /api/notes/<id>  → 删除笔记
```
